# EcoLivros

Frontend da plataforma EcoLivros, desenvolvido com React e Vite.

## Sprint 1

- Configuracao inicial do projeto.
- Layout base com Navbar e Footer.
- Tela Home responsiva.
- Componentes reutilizaveis iniciais.

## Sprint 2

- React Router configurado.
- Rotas iniciais para Home, Login e Cadastro.
- Formularios responsivos de autenticacao.
- Componente Input reutilizavel.

## Sprint 3

- Feed de livros disponiveis.
- Tela de pesquisa com filtro por texto.
- Cards de livros reaproveitados em Home, Feed e Pesquisa.
- Camada inicial de service mockado para livros.

## Sprint 4

- Cadastro de livro.
- Campos de formulario para titulo, autor, categoria, localizacao, tipo, estado e descricao.

## Sprint 5

- Tela de detalhes do livro.
- Tela de perfil com resumo de participacao e livros ativos.

## Sprint 6

- Tela de chat com contatos, historico de mensagens e campo de envio.

## Sprint 7

- Ajustes responsivos para telas novas.
- Reutilizacao de componentes e service de livros.

## Sprint 8

- Pagina 404.
- Configuracao de rewrite para Vercel.
- Validacao de build.

## Como executar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
